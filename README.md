# Sistema Biblioteca Java

Projeto desenvolvido para praticar Programação Orientada a Objetos (POO) em Java.

## Funcionalidades

- Cadastrar livros
- Listar livros cadastrados
- Encerrar o sistema

## Conceitos utilizados

- Classes
- Objetos
- Construtores
- ArrayList
- Scanner
- Encapsulamento
- Laços de repetição
- Estruturas de decisão
